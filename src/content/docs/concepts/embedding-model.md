# Mô hình nhúng (Embedding Model)

## Summary
Mô hình AI chịu trách nhiệm chuyển hóa ngôn ngữ tự nhiên thành dạng vector đa chiều nhằm phục vụ tìm kiếm ngữ nghĩa.

## Key Characteristics
- Chuyển văn bản đầu vào thành một mảng số thực.
- Cần đồng bộ mô hình nhúng lúc lưu trữ và lúc truy vấn câu hỏi người dùng.
- Độ dài vector phổ biến thường từ 384, 768 đến 1536 chiều.

*(Bài viết đang trong quá trình hoàn thiện. Phiên bản đầy đủ sẽ được cập nhật sớm)*
