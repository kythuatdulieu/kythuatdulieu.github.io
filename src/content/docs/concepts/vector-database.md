# Cơ sở dữ liệu Vector (Vector Database / Vector Store)

## Summary
Hệ thống cơ sở dữ liệu chuyên biệt để lưu trữ và truy vấn nhanh các vector đặc trưng (embeddings) của dữ liệu không cấu trúc như văn bản, hình ảnh, âm thanh.

## Key Characteristics
- Cho phép thực hiện tìm kiếm ngữ nghĩa (semantic search) bằng cách đo khoảng cách cosine hoặc L2 giữa các vector.
- Các công cụ phổ biến: Databricks Vector Search, Pinecone, Chroma, Milvus, pgvector.
- Là thành phần cốt lõi của các ứng dụng RAG để lưu trữ các phân đoạn tài liệu đã được nhúng.
