# Học qua vài ví dụ (Few-shot Prompting)

## Summary
Kỹ thuật hướng dẫn LLM bằng cách đưa ra một vài ví dụ minh họa về cặp (câu hỏi - câu trả lời mẫu) ngay trong prompt đầu vào trước khi yêu cầu nó thực hiện nhiệm vụ thực tế.

## Key Characteristics
- Giúp LLM hiểu rõ định dạng đầu ra phức tạp hoặc logic suy luận cụ thể mà mô hình khó nắm bắt qua mô tả bằng chữ.
- Rất hiệu quả cho các tác vụ phân loại dữ liệu, trích xuất thực thể hoặc tạo mã nguồn.
- Khác với tinh chỉnh (fine-tuning) vì không hề thay đổi trọng số mô hình mà chỉ tận dụng khả năng học ngữ cảnh (in-context learning).

*(Bài viết đang trong quá trình hoàn thiện. Phiên bản đầy đủ sẽ được cập nhật sớm)*
