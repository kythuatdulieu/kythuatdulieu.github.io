# Cơ sở dữ liệu Vector (Vector Database / Vector Store)

## Summary
Nơi lưu trữ các vector nhúng (embeddings) đại diện cho nội dung các phân đoạn văn bản, cho phép thực hiện truy xuất nhanh dựa trên độ tương đồng ngữ nghĩa.

## Key Characteristics
- Chuyển đổi văn bản thành biểu diễn số học đa chiều.
- Đóng vai trò như bộ nhớ ngoài cho hệ thống RAG để tìm kiếm thông tin liên quan.
- Hỗ trợ tìm kiếm theo khoảng cách (Cosine, Euclidean) để tìm đoạn văn bản gần nhất với câu hỏi.
