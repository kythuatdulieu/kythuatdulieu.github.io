# Phục vụ mô hình (Model Serving)

## Summary
Quá trình triển khai các mô hình Machine Learning hoặc LLM dưới dạng các API endpoint có độ trễ thấp, tính sẵn sàng cao để các ứng dụng khác có thể gọi và sử dụng.

## Key Characteristics
- Databricks Model Serving cung cấp môi trường serverless được quản lý hoàn toàn cho việc chạy suy luận mô hình.
- Hỗ trợ tự động mở rộng quy mô (auto-scaling) dựa trên lưu lượng truy cập thực tế, giúp tối ưu hóa chi phí.
- Cho phép phân phối cả các mô hình mã nguồn mở (như Llama, Mistral) và các mô hình độc quyền.

*(Bài viết đang trong quá trình hoàn thiện. Phiên bản đầy đủ sẽ được cập nhật sớm)*
