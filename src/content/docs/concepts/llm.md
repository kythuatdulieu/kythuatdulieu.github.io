# Large Language Model (Mô hình Ngôn ngữ Lớn)

## Summary
Các mô hình học sâu (deep learning) có quy mô hàng tỷ tham số, được huấn luyện trên lượng dữ liệu văn bản khổng lồ để hiểu, tạo và xử lý ngôn ngữ tự nhiên như con người.

## Key Characteristics
- Hoạt động dựa trên kiến trúc Transformer nổi tiếng.
- Ví dụ phổ biến: GPT-4, Llama 3, DBRX, Claude, Gemini.
- Có khả năng thực hiện nhiều tác vụ như dịch thuật, tóm tắt, lập trình và hội thoại.
