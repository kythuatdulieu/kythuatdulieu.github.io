# Token (Đơn vị từ vựng)

## Summary
Các phân đoạn văn bản nhỏ nhất (có thể là một từ, một phần của từ hoặc thậm chí là một ký tự) mà mô hình ngôn ngữ lớn (LLM) sử dụng để xử lý và tạo văn bản.

## Key Characteristics
- Trung bình, 1 token tương đương khoảng 4 ký tự hoặc 0.75 từ tiếng Anh.
- Chi phí sử dụng API của các nhà cung cấp LLM thường được tính dựa trên số lượng token đầu vào (prompt) và đầu ra (completion).
- Việc quản lý số lượng token (token count) giúp kiểm soát chi phí và tránh vượt quá giới hạn cửa sổ ngữ cảnh.

*(Bài viết đang trong quá trình hoàn thiện. Phiên bản đầy đủ sẽ được cập nhật sớm)*
