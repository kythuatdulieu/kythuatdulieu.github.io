# Project Brief: kythuatdulieu

## Project Overview
**kythuatdulieu** is a blog website built with Astro, designed to be deployed on GitHub Pages. The project serves as a platform for sharing content related to data science and technical topics.

## Core Requirements
- Static site generation using Astro
- GitHub Pages deployment
- Blog functionality with MDX support
- SEO optimization with sitemap
- RSS feed support
- Responsive design

## Technical Stack
- **Framework**: Astro 5.11.0
- **Content**: MDX for enhanced markdown
- **Deployment**: GitHub Pages via GitHub Actions
- **Styling**: Minimal CSS (customizable)
- **Build**: Static site generation

## Project Goals
1. Create a fast, SEO-friendly blog
2. Deploy automatically via GitHub Actions
3. Maintain 100/100 Lighthouse performance
4. Support both markdown and MDX content
5. Provide RSS feed for content syndication
