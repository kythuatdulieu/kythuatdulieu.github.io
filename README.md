### 📖 Về Chúng Tôi (About)
[**kythuatdulieu.github.io**](https://kythuatdulieu.github.io) là dự án mã nguồn mở được xây dựng nhằm chuẩn hóa và hệ thống hóa kiến thức **Modern Data Stack**. Chúng tôi tập trung vào kiến trúc Lakehouse, Data Pipeline, Data Quality, Orchestration và GenAI.

🌟 **Điểm Nổi Bật (Key Features):**
- **Trải Nghiệm Đọc Liền Mạch (Popover Backlinks):** Tích hợp tính năng popover tự động tra cứu thuật ngữ chuyên ngành (PACELC, CAP, ETL...) ngay trên văn bản đang đọc. Mạch tư duy của bạn sẽ không bị ngắt quãng bởi việc phải mở tab mới.
- **Hệ Thống Kiến Thức Kỹ Thuật:** Kiến thức được tổ chức theo luồng kỹ thuật thực tế: Hệ thống phân tán, Lưu trữ, Xử lý lô/luồng, Mô hình hóa, DataOps và GenAI.
- **Chứng Chỉ & Phỏng Vấn:** Cung cấp không gian luyện thi (Quiz) với bộ câu hỏi bám sát các kỳ thi chứng chỉ Databricks, Snowflake, AWS... cùng ngân hàng câu hỏi phỏng vấn System Design.

### 🏗️ Kiến Trúc Nội Dung (Content Architecture)
Sổ tay bao quát các lĩnh vực cốt lõi trong Data Engineering:

1. **Khái Niệm Cốt Lõi (Concepts):** Đi sâu vào nguyên lý hoạt động của các thành phần trong Modern Data Stack.
2. **Lộ Trình Sự Nghiệp (Learning Paths):** Hướng dẫn phát triển bộ kỹ năng theo từng cấp độ và chuyên môn.

5. **Chứng Chỉ (Quizzes):** Khu vực thi thử và đánh giá kiến thức.

### 💻 Dành Cho Kỹ Sư Đóng Góp (Development)

Dự án sử dụng framework ưu tiên hiệu năng tài liệu: **[Astro](https://astro.build/) + [Starlight](https://starlight.astro.build/)**.
Hỗ trợ render sơ đồ kiến trúc với `Mermaid.js` và công thức toán học với `KaTeX`.

#### Yêu cầu hệ thống
- [Node.js](https://nodejs.org/) (Khuyến nghị LTS >= 18)
- Git

#### Cài đặt và Chạy cục bộ

```bash
# Clone kho lưu trữ
git clone https://github.com/kythuatdulieu/kythuatdulieu.github.io.git
cd kythuatdulieu.github.io

# Cài đặt các gói phụ thuộc
npm install

# Khởi động môi trường phát triển (http://localhost:4321)
npm run dev
```

#### Cấu trúc thư mục chính
- `/src/content/docs`: Nơi chứa toàn bộ bài viết (định dạng `.md` hoặc `.mdx`).
- `/src/assets`: Chứa hình ảnh và tài nguyên tĩnh.
- `astro.config.mjs`: Tệp cấu hình giao diện, menu điều hướng (sidebar), và các plugin.
